# Strategic Roadmap
HostelPulse development path for 2025/2026.

## Phase 1: Operation Bedrock (Current)
*Target: Q4 2025*
- Restore core hostel operations (Rooms, Guests, Bookings).
- Establish high-quality UI/UX standards.
- 100% test coverage for critical business actions.

## Phase 2: Commercial Readiness
*Target: Q1 2026*
- **Subscriptions:** Integration with Stripe for Team billing.
- **Reporting:** Advanced revenue and occupancy charts.
- **Settings:** Property-specific custom branding and invoice generation.

## Phase 3: AI & Automation
*Target: Q2 2026*
- **Agno Re-integration:** Bring back the AI Assistant for guest communications.
- **Smart Pricing:** Dynamic pricing recommendations based on local events.
- **OCR Import:** Automatic guest ID scanning and data entry.

## Phase 4: Ecosystem
*Target: Q3 2026+*
- **Channel Manager:** API integrations with Booking.com and Hostelworld.
- **Mobile App:** React Native companion for staff.
