## Phase 2: Commercial Readiness

_Target: Q1 2026_

- **Subscriptions:** Integration with Stripe for Team billing.
- **Reporting:** Advanced revenue and occupancy charts.

_Target: Q2 2026_

- **Agno Re-integration:** Bring back the AI Assistant for guest communications.

- **OCR Import:** Automatic guest ID scanning and data entry.

## Phase 4: Ecosystem

_Target: Q3 2026+_

- **Channel Manager:** API integrations with Booking.com and Hostelworld.
- **Mobile App:** React Native companion for staff.
